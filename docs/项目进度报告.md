## 四、经验教训

### 1. 技术选型
- ❌ **JoyAgent**: Java技术栈与Python混合，维护成本高，代码质量差
- ✅ **纯Python方案**: 技术栈统一，维护简单，完全可控

### 2. MCP协议理解
- MCP (Model Context Protocol) 是AI与工具交互的标准协议
- 需要正确实现工具定义和执行逻辑
- 工具描述要清晰，让AI理解如何使用

### 3. 流式响应处理
- SSE (Server-Sent Events) 需要正确处理
- 前端需要正确解析流式数据
- React StrictMode会导致组件渲染两次

### 4. 错误处理
- 需要完善的错误处理和日志
- AI Agent需要能够处理工具调用失败的情况
- 用户需要看到清晰的错误信息
