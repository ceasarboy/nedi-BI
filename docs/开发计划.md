## 二、当前架构

```
┌─────────────────────────────────────────────────────────────┐
│                        前端 (React + Vite)                   │
│  配置管理 │ 数据查询 │ 数据分析 │ 可视化 │ AI对话 │ 设置    │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    PB-BI 后端 (Python FastAPI)               │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │  数据流引擎  │  │  MCP 工具   │  │  AI Agent   │         │
│  │             │  │  (19种图表) │  │             │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │  对话管理    │  │  设置管理    │  │  记忆系统    │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                      数据层                                  │
│   SQLite │ 明道云API │ 其他数据源                           │
└─────────────────────────────────────────────────────────────┘
```
