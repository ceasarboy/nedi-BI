# PB-BI 数据流配置技能

## 触发场景
当用户提到"配置数据流"、"添加数据源"、"明道云数据"、"本地数据导入"、"字段配置"、"数据流管理"等关键词时使用此技能。

## 功能概述
PB-BI支持两种数据源类型：明道云API数据源和本地文件数据源。本技能指导如何正确配置和管理数据流。

---

## 一、明道云数据流配置

### 1. 创建明道云数据流
1. 进入配置管理页面（首页）
2. 点击"增加明道云数据流"按钮
3. 填写数据流信息：
   - **名称**：数据流的显示名称
   - **AppKey**：明道云应用的AppKey
   - **Sign**：明道云应用的Sign
   - **WorksheetId**：工作表ID

### 2. API请求头配置 ⭐重要
```python
headers = {
    "Content-Type": "application/json",
    "HAP-Appkey": appkey,  # 注意是 HAP-Appkey 不是 AppKey
    "HAP-Sign": sign        # 注意是 HAP-Sign 不是 Sign
}
```
**关键点**：请求头必须使用 `HAP-Appkey` 和 `HAP-Sign`（大小写敏感）

### 3. API端点使用
- 获取应用信息：`GET /v3/app`
- 获取工作表字段：`GET /v3/app/worksheets/{worksheet_id}`
- 查询记录：`POST /v3/app/worksheets/{worksheet_id}/rows/list`

**注意**：不要使用 `/v3/app/info`，应该使用 `/v3/app`

### 4. 字段配置
1. 点击数据流的"设置"按钮
2. 系统自动获取明道云字段
3. 配置每个字段的：
   - **字段类型**：文本、数字、日期、布尔等
   - **启用状态**：勾选启用/禁用该字段
4. 点击"保存配置"

## 二、本地数据流配置

### 1. 创建本地数据流
1. 进入配置管理页面
2. 点击"增加本地数据流"按钮
3. 填写数据流名称
4. 点击"保存"

### 2. 导入文件获取字段
1. 点击本地数据流的"设置"按钮
2. 点击"导入文件获取字段"
3. 选择CSV或Excel文件
4. 系统自动解析字段
5. 配置字段类型和启用状态
6. 点击"保存配置"

### 3. 支持的文件格式
- **CSV**：逗号分隔值文件
- **Excel**：.xlsx、.xls格式

## 四、常见问题

### 1. 明道云连接失败
**检查项**：
- AppKey和Sign是否正确
- WorksheetId是否存在
- 网络是否可访问明道云API
- 请求头是否使用正确的字段名

### 2. 字段获取失败
**检查项**：
- API返回格式是否正确
- 字段ID可能是`fieldId`或`id`，需兼容

### 3. 本地文件导入失败
**检查项**：
- 文件格式是否支持
- 文件编码是否正确（CSV推荐UTF-8）
- 文件是否有表头

## 六、经验教训

1. **请求头大小写敏感**：明道云API要求`HAP-Appkey`和`HAP-Sign`，不能使用其他格式
2. **字段类型影响分析**：正确配置字段类型可以提升后续分析和可视化效果
3. **本地数据流流程**：先创建数据流，再导入文件获取字段，与明道云流程一致
4. **字段启用状态**：禁用的字段不会在数据查询和分析中显示
